Go SDK Installation
==================

Requirements
------------

* Go >= 1.16

Installation
------------

.. code-block:: bash

    go get github.com/nexacon/nexacon-go-sdk

Verification
------------

Verify the installation by importing the SDK:

.. code-block:: go

    package main

    import "github.com/nexacon/nexacon-go-sdk"
    import "fmt"

    func main() {
        fmt.Println("Nexacon SDK installed successfully")
    }
