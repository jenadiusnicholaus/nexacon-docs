Ruby SDK Installation
=====================

Requirements
------------

* Ruby >= 2.7
* Bundler

Installation
------------

Add to your `Gemfile`:

.. code-block:: ruby

    gem 'nexacon-sdk'

Then run:

.. code-block:: bash

    bundle install

Verification
------------

Verify the installation by importing the SDK:

.. code-block:: ruby

    require 'nexacon-sdk'
    puts 'Nexacon SDK installed successfully'
